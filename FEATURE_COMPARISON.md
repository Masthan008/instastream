## Missing Features — Priority for Implementation

### P0 (High Impact, High Value)
1. **YouTube Search built-in** — Search YouTube without leaving the app
2. **Download progress notifications** — Shows progress in Android notification bar
3. **Resume interrupted downloads** — Resume failed downloads

### P1 (Medium Impact)
4. **Smart Mode** — Remember download format/quality preferences
5. **Direct save to device gallery** — Save directly to phone's media store
6. **Background downloads** — Continue downloading when app is backgrounded

### P2 (Nice to Have)
7. **YouTube Shorts URL detection** — Handle `youtube.com/shorts/` URLs
8. **Gallery search/filter** — Search downloaded files by name
9. **Local playlist management** — Create playlists from downloaded files
10. **Picture-in-Picture mode** — Floating video player
11. **Download to SD card** — Configurable save location
12. **Cache management** — Clear temp files from settings
13. **Audio equalizer** — EQ presets for audio player
